# ПЛЮС Вектор

Браузерний векторний редактор для шкільного офісного пакета.

## Можливості

- SVG-полотно без бекенду.
- Інструменти: вибір, олівець, лінія, стрілка, прямокутник, еліпс, трикутник, ромб, зірка, текст.
- Переміщення, виділення, зміна розміру і редагування тексту.
- Контур, заливка, товщина, прозорість і розмір шрифту.
- Підкладка: немає, сітка, рядки.
- Прив'язка до сітки.
- Undo / redo, дублювання, видалення і порядок шарів.
- Збереження проєкту в JSON.
- Експорт у SVG та PNG.
- Друк і автозбереження чернетки в localStorage.

Вектор підключений до спільного shell-шару офісу і має дотримуватися глобальних стандартів:

- [../UI_INTEGRATION_GUIDE.md](../UI_INTEGRATION_GUIDE.md)
- [UI_STANDARD.md](UI_STANDARD.md)
- [UI_MIGRATION_TO_STANDARD.md](UI_MIGRATION_TO_STANDARD.md)

Стандартні команди меню і панелі мають іти через `OfficeUI.registerCommands` / `OfficeUI.runCommand`, а відкриття файлів через `OfficeUI.openFilePicker`.

## Гарячі клавіші

- `V` — вибір
- `P` — олівець
- `L` — лінія
- `A` — стрілка
- `R` — прямокутник
- `O` — еліпс
- `D` — ромб
- `S` — зірка
- `T` — текст
- `Ctrl+Z` / `Ctrl+Y` — скасувати / повернути
- `Ctrl+D` — дублювати
- `Ctrl+C` / `Ctrl+V` — копіювати / вставити
- `Ctrl+S` — зберегти проєкт
- `Ctrl+Shift+S` — експортувати PNG
- `Delete` / `Backspace` — видалити
