# KEYBOARD_SHORTCUTS.md

Статус: активне джерело правди.

## Глобальні комбінації

| Дія | Комбінація |
|---|---|
| Новий | Ctrl+N |
| Відкрити | Ctrl+O |
| Зберегти | Ctrl+S |
| Скасувати | Ctrl+Z |
| Повернути | Ctrl+Y або Ctrl+Shift+Z, якщо редактор підтримує |
| Виділити все | Ctrl+A |
| Копіювати | Ctrl+C |
| Вставити | Ctrl+V |
| Вирізати | Ctrl+X |
| Закрити верхній overlay | Esc |

## Контракт

- `new/open/save/undo/redo` мають маршрутизуватись через `OfficeShell.runCommand` або `OfficeUI.runCommand`.
- Комбінації, показані в меню, мають реально працювати.
- Якщо редактор має виняток, він має бути очевидним у меню або title.
- Hotkeys не мають спрацьовувати всередині input/textarea/select, якщо це ламає введення тексту.

## Поточний допустимий виняток

У `flowcharts` `Ctrl+S` використовується для PNG export, а збереження JSON-проєкту відповідає toolbar Save і `Ctrl+Shift+S`.
