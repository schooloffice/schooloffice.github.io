# Інструкція для агента

Репозиторій `schooloffice.github.io` є самостійним проєктом Офіс ПЛЮС. Старе обмеження працювати лише в підкаталозі `office/` більше не актуальне.

Перед роботою прочитай:

- `PROMPT_FOR_AGENT.md`
- `PROJECT_DIRECTION.md`
- `README.md`
- документацію редактора, якого стосується завдання

Працюй у межах цього репозиторію, зберігай shared shell, офлайн-режим і продуктову мету локального шкільного пакета редакторів.
