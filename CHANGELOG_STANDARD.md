# CHANGELOG_STANDARD.md

Статус: довідковий шаблон.

Фактичний журнал змін пакета ведеться в `CHANGELOG.md`.

Рекомендований формат запису:

```md
## YYYY-MM-DD

### Area

- Що змінено.
- Який контракт або тест оновлено.
- Яку перевірку виконано.
```

Для системних UI-змін обов'язково зазначай:

- чи змінено `office-ui.js`;
- чи змінено статичний аудит;
- чи зачеплено command/file-picker/modal/dropdown/status contracts;
- чи запускались браузерні тести, або чому вони не потрібні.

Не дублюй тут фактичний changelog.
